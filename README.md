# Tera — U.S. market site

**Tera** is a small studio that designs and builds **custom websites, apps, and internal tools** for businesses across the United States. We work with owners and operators who are done bending generic software to fit how they actually run—construction crews, clinics, home services, logistics, and more.

This repository is the **public marketing site** for Tera’s U.S. offering: a single, fast landing experience that explains **what we build**, **how we work**, **selected client work**, and a **straightforward way to start a conversation**.

**Live site:** [buildwithtera.com](https://buildwithtera.com)

---

### What you’ll find here

- **Clear positioning** — portals, dashboards, field tools, and marketing sites scoped in writing before we ship code.  
- **Proof, not fluff** — case-style proof, portfolio entries, and a real client voice in the testimonial.  
- **Human contact** — project form powered by [Formspree](https://formspree.io), plus a dedicated privacy page describing how inquiries are handled.

Built with **Next.js**, **Tailwind CSS**, and **TypeScript**, optimized for performance and accessibility on real devices.
